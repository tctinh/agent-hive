# Release v1.3.3

Compare: [`v1.3.2...v1.3.3`](https://github.com/tctinh/agent-hive/compare/v1.3.2...v1.3.3)

---

## Highlights

### PR #57 Recovery Lands for Real

This corrective patch release restores the missing GitHub Copilot rewrite from PR #57 onto the published `release/human-overview-context-v1` baseline. The recovered payload includes the generator-based Copilot artifacts, updated language-model tool wiring, regenerate-agents command flow, and the intended VS Code extension/sidebar cleanup that should have been present on the `1.3.x` line.

**Why it matters:** The original `v1.3.2` release notes claimed this payload shipped, but the published branch history showed it had not. `v1.3.3` is the first tag where the recovered PR #57 functionality is actually present.

### Release History and Lockfile Repaired

This release also corrects the historical release narrative and the lockfile damage introduced by the earlier version/doc rewrite. The `v1.3.2` notes now describe what truly shipped there, and `package-lock.json` has been regenerated from corrected manifests so workspace version bumps no longer corrupt the third-party `once` entry.

**Why it matters:** Users can trust both the release documentation and the published dependency metadata again. The recovery fixes the product and the history describing it.
